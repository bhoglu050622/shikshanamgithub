## 🐛 Common Issues to Check

1. **Hydration Errors**
   - All animations wrapped in HydrationSafeMotion
   - No client-only code in server components
   - Proper 'use client' directives

2. **Image Loading**
   - RobustImage components with fallbacks
   - Proper sizes attributes
   - Priority loading on hero images

3. **Animation Conflicts**
   - No competing animations
   - Proper z-index layering
   - Overflow handling

4. **Dark Mode**
   - All gradients have dark variants
   - Text contrast sufficient
   - Borders visible in dark mode
